## 边界判断

V0.27 是工业级 agent 的地基，不是完整产品级 agent。它证明生成过程已经从“单函数生成”推进到“可观测图式状态机”，但还没有完成：

- LangGraph runtime 持久化与 checkpoint。
- 多 agent 协作。
- LLM 生成章节/场景/状态变量/结局矩阵。
- 语义级 repair。
- human-in-the-loop 审稿节点。

下一轮应继续把 deterministic skeleton 中的一部分拆成真实 agent node，例如章节规划 node 或状态变量设计 node，并保留同样的 schema/validator/content QA gates。
